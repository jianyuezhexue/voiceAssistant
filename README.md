# VoiceAssistant

智能语音输入助手 - 让会议讨论自动转化为结构化的代办事项和知识库

## 项目简介

VoiceAssistant 是一款智能语音助手系统，旨在帮助用户在会议讨论中自动整理代办事项和业务知识，实现从语音输入到知识管理的完整闭环。通过阿里云实时语音识别和大模型智能解析，将口语化的讨论内容转化为结构化的代办项和知识点，支持后续的知识检索与回溯。

## 核心功能

### 语音识别与解析
- **实时语音转文字**：基于阿里云 ASR 服务，实时将用户语音转换为文字
- **智能内容解析**：利用大模型对识别内容进行语义理解，自动分类识别：
  - 日程安排
  - 待办事项
  - 知识内容

### 数据存储与管理
- **待办事项存储**：代办项存入 MySQL，支持完整的 CRUD 操作
- **知识库存储**：知识点存入 Milvus 向量数据库，支持语义检索
- **缓存层**：Redis 作为缓存层，提升系统响应速度

### 前端交互
- **代办项展示**：实时回显代办事项列表
- **知识点展示**：展示提取的业务知识点
- **知识检索**：支持语义化检索某一知识点

### 完整闭环
```
会议讨论 → 语音识别 → 内容解析 → 分类存储 → 页面展示 → 知识回溯
```

## 快速开始

### 环境要求

- Go 1.21+
- Node.js 18+
- MySQL 8.0+
- Redis 6.0+
- Milvus 2.3+

### 安装依赖

```bash
# 后端依赖
go mod download

# 前端依赖
cd web && npm install
```

### 配置文件

复制配置模板并填写必要的配置：

```bash
cp config/config.example.yaml config/config.yaml
```

主要配置项：

```yaml
server:
  port: 8080

database:
  mysql:
    host: localhost
    port: 3306
    user: root
    password: your_password
    database: voice_assistant
  redis:
    host: localhost
    port: 6379
  milvus:
    host: localhost
    port: 19530

asr:
  provider: aliyun
  app_key: your_app_key
  access_key_id: your_access_key_id
  access_key_secret: your_access_key_secret

llm:
  provider: claude
  api_key: your_api_key
  model: claude-sonnet-4-6
```

### 启动服务

```bash
# 启动后端服务
go run cmd/server/main.go

# 启动前端服务
cd web && npm run dev
```

## API 文档

### 语音识别

```
WebSocket /ws/asr
- 建立实时语音识别连接
- 支持流式音频传输
```

### 待办事项

```
GET    /api/todos          # 获取待办列表
POST   /api/todos          # 创建待办
PUT    /api/todos/:id      # 更新待办
DELETE /api/todos/:id      # 删除待办
```

### 知识库

```
POST   /api/knowledge      # 添加知识点
GET    /api/knowledge      # 获取知识列表
POST   /api/knowledge/search  # 语义检索知识点
DELETE /api/knowledge/:id  # 删除知识点
```

## 数据流

1. **语音输入**：用户通过前端进行语音输入
2. **实时转写**：音频流通过 WebSocket 发送到后端，调用阿里云 ASR 实时转写
3. **内容解析**：转写文本交由大模型进行语义理解和分类
4. **数据存储**：
   - 待办事项 → MySQL
   - 知识内容 → Milvus（向量嵌入后存储）
5. **前端回显**：实时展示解析结果
6. **知识回溯**：用户可通过语义检索查询历史知识点

## 阿里云 ASR 配置

本项目使用阿里云实时语音识别服务，需要开通以下服务：

1. 阿里云智能语音服务
2. 创建 ASR 应用获取 AppKey

[阿里云语音识别文档](https://help.aliyun.com/product/30413.html)
